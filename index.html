Sécurité
Gérez les accès administrateur
🔑 Changer le mot de passe

Exigences : 12 caractères minimum, majuscule + minuscule + chiffre

👤 Identifiants & Sécurité

Utilisateur : admin
Le mot de passe est stocké sous forme de hash SHA-256 — jamais en clair.
⚠️ Changez le mot de passe par défaut dès la première connexion.

🛡️ Statut de sécurité
✓ Mot de passe haché SHA-256
✓ Blocage après 5 tentatives (30s)
✓ Session expirée après 60 min d'inactivité
✓ Token GitHub jamais exposé dans le code source
⚠ Authentification côté client (limitation inhérente)

Notre mission,
bâtir demain

Le Club Génie Civil de la Polytechnique de Diamniadio réunit étudiants, ingénieurs et chercheurs autour de la passion commune de construire des infrastructures durables et innovantes.
